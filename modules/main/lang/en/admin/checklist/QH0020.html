You have to configure the server software manually for optimum performance.

To achieve best results, use the Bitrix Environment package (for Windows or Linux), or Bitrix Virtual Appliance. Alternatively, you may ask your hosting provider to configure the server for you.

Generally, the more the expected website traffic, the higher performance tariff is required. It is recommended that you perform a stress test on the content of expected magnitude.
